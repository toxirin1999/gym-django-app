# GYM_PLAN_V2 — Prueba de campo (Phase 47)

> El sistema ya pasó tests. Ahora tiene que pasar una semana real sin pedir más atención de la que devuelve.

**Regla de Phase 47:**
Durante la prueba, no se corrige nada salvo bugs evidentes. Se observa.
Si cambias el sistema cada día, no sabrás si funciona o si simplemente persigues sensaciones.

## Clasificación de hallazgos

| Tipo | Ejemplo | Acción |
|---|---|---|
| **Bug real** | `vigilar_senal` afectó cargas | Corregir inmediatamente |
| **Problema de UX** | Centro bien pero muy abajo en móvil | → Phase 48 |
