CÓMO CONFIGURAR EL FLUJO COMPLETO
1. Apple Watch → Strava (una vez)
En iPhone: Strava → Configuración → Aplicaciones → Conectar Apple Health.

2. Strava → Este servidor (webhook)
Registra el webhook en Strava para: https://tu-dominio.com/hyrox/strava/webhook/
Usa el verify token configurado en tu .env como STRAVA_VERIFY_TOKEN.

3. Flujo automático
Grabas → Apple Watch sincroniza → Strava notifica al servidor → aparece aquí para confirmar.
