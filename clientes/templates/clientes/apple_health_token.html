◆ Cómo configurar el Atajo en iOS

{% for paso in pasos %}{% endfor %}
1
Abre la app Atajos en tu iPhone
Toca el botón "+" para crear un nuevo atajo
2
Añade "Buscar entrenamientos"
Configurar: Límite = 1, Ordenar por = Fecha de inicio más reciente
3
Añade "Obtener detalles del entrenamiento"
Selecciona el entrenamiento del paso anterior. Repite para cada dato: Duración, Calorías activas, FC media, FC máxima, Tipo de actividad, Fecha de inicio
4
Añade "Obtener contenido de la URL"
URL: {{ endpoint }}
Método: POST · Tipo: JSON
Cuerpo:
{
  "token": "{{ token }}",
  "workout_type": [Tipo de actividad],
  "start_date": [Fecha de inicio],
  "duration_minutes": [Duración en minutos],
  "calories": [Calorías activas],
  "avg_heart_rate": [FC media],
  "max_heart_rate": [FC máxima]
}
5
Guarda el atajo y ejecútalo tras cada entreno
También puedes añadirlo a la pantalla de inicio del Apple Watch para ejecutarlo directamente desde la muñeca
