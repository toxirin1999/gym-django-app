💡 ¿Por qué las 4 Leyes?

Las 4 Leyes de Atomic Habits son un framework científico para diseñar hábitos que se mantengan. Cada ley aborda un aspecto clave del comportamiento humano.

1️⃣

{% if habito.tipo_habito == 'positivo' %} Hacerlo Obvio {% else %} Hacerlo Invisible {% endif %}

{% if habito.tipo_habito == 'positivo' %} Un hábito que no ves, no existe. Diseña tu entorno para que la señal sea imposible de ignorar. {% else %} Fuera de vista, fuera de mente. Elimina las señales que disparan este hábito. {% endif %}

💡 Ejemplos:
{% if habito.tipo_habito == 'positivo' %}
🏃 Ejercicio: Deja tu ropa deportiva lista la noche anterior
📚 Leer: Coloca un libro en tu almohada cada mañana
💧 Beber agua: Ten una botella siempre visible en tu escritorio
{% else %}
🚬 Fumar: Esconde los cigarrillos en un lugar difícil de alcanzar
📱 Redes sociales: Desinstala las apps de tu teléfono
🍪 Comer mal: No compres snacks poco saludables
{% endif %}
2️⃣

{% if habito.tipo_habito == 'positivo' %} Hacerlo Atractivo {% else %} Hacerlo No Atractivo {% endif %}

{% if habito.tipo_habito == 'positivo' %} Asocia el hábito con algo que ya disfrutas. Usa el "temptation bundling" a tu favor. {% else %} Resalta las consecuencias negativas. Haz que el hábito pierda su atractivo. {% endif %}

💡 Ejemplos:
{% if habito.tipo_habito == 'positivo' %}
🏃 Ejercicio: Escucha tu podcast favorito solo mientras corres
📚 Leer: Lee en tu café favorito con un buen café
🥗 Comer sano: Cocina con tu música favorita
{% else %}
🚬 Fumar: Visualiza tus pulmones negros cada vez que sientes el impulso
📱 Procrastinar: Calcula cuánto dinero pierdes por hora perdida
🍔 Comida basura: Mira documentales sobre procesamiento de alimentos
{% endif %}
3️⃣

{% if habito.tipo_habito == 'positivo' %} Hacerlo Fácil {% else %} Hacerlo Difícil {% endif %}

{% if habito.tipo_habito == 'positivo' %} Reduce la fricción. Aplica la regla de los 2 minutos: empieza con la versión más simple. {% else %} Aumenta la fricción. Añade pasos extra entre tú y el hábito negativo. {% endif %}

💡 Ejemplos:
{% if habito.tipo_habito == 'positivo' %}
🏃 Ejercicio: "Solo me pongo las zapatillas" (regla 2 min)
📚 Leer: "Solo leo una página"
🧘 Meditar: "Solo respiro 3 veces profundamente"
{% else %}
🚬 Fumar: Guarda los cigarrillos en el sótano, no en el bolsillo
📱 Redes sociales: Usa apps que bloqueen el acceso durante horas de trabajo
🎮 Videojuegos: Desenchufa la consola después de cada sesión
{% endif %}
4️⃣

{% if habito.tipo_habito == 'positivo' %} Hacerlo Satisfactorio {% else %} Hacerlo Insatisfactorio {% endif %}

{% if habito.tipo_habito == 'positivo' %} Recompénsate inmediatamente. Lo que se recompensa, se repite. {% else %} Crea consecuencias inmediatas. El dolor debe ser instantáneo. {% endif %}

💡 Ejemplos:
{% if habito.tipo_habito == 'positivo' %}
🏃 Ejercicio: Marca un ✓ en un calendario visible
📚 Leer: Mueve una canica a un frasco por cada página
💰 Ahorrar: Transfiere $1 a tu cuenta de ahorros cada día que lo logres
{% else %}
🚬 Fumar: Dona $10 a una causa que odias cada vez que recaigas
📱 Procrastinar: Haz 10 burpees cada vez que abras redes sociales
🍔 Comida basura: Comparte públicamente cada recaída en redes
{% endif %}
